## 📊 Column Descriptions

### 1. File Name
- **Type**: String (bold)
- **Purpose**: Identify the quarantined file
- **Example**: `malware.exe`, `trojan.dll`
- **Styling**: Bold font weight

### 2. Quarantine Count
- **Type**: Integer
- **Purpose**: Total times this file was quarantined across all hosts/months
- **Example**: `15`, `12`, `8`
- **Alignment**: Center

### 3. Affected Hosts
- **Type**: Integer
- **Purpose**: Number of unique hosts where this file was detected
- **Example**: `5`, `3`, `1`
- **Alignment**: Center

### 4. Hostname(s)
- **Type**: String (comma-separated list)
- **Purpose**: Show which specific hosts were affected
- **Format**:
  - Shows first 5 hostnames
  - If more than 5: adds `(+X more)` at the end
- **Example**:
  - `DESKTOP-001, LAPTOP-002, PC-003`
  - `DESKTOP-001, LAPTOP-002, PC-003, SERVER-004, WS-005 (+3 more)`
- **Styling**: Smaller font (9px) to fit long lists
- **Alignment**: Left

### 5. First Seen
- **Type**: Date/Time
- **Purpose**: When this file was first quarantined
- **Format**: `DD/MM/YYYY HH:MM`
- **Example**: `01/02/2026 10:30`
- **Styling**: 9px font
- **Alignment**: Center

### 6. Last Seen
- **Type**: Date/Time
- **Purpose**: When this file was most recently quarantined
- **Format**: `DD/MM/YYYY HH:MM`
- **Example**: `23/02/2026 14:45`
- **Styling**: 9px font
- **Alignment**: Center

## 📈 Data Source

The table data comes from:
- **Source**: `quarantine_data['detailed_file_summary']`
- **Processing**: Generated by `generate_quarantine_analysis()` function
- **Sorting**: Files sorted by quarantine count (descending)
- **Limit**: Top 10 files shown in PDF
- **Full data**: Available in Pivot Table Builder for deeper analysis

---

## 🔍 How Data is Calculated

### Per File:
1. **Quarantine Count**:
   - Count total rows in JSON where `File Name` matches
   - Includes all occurrences across all months

2. **Affected Hosts**:
   - Count unique `Hostname` values for this file
   - Uses `nunique()` function

3. **Hostname(s) List**:
   - Get all unique hostnames for this file
   - Take first 5 names
   - If more than 5: append `(+X more)` where X = total - 5

4. **First Seen**:
   - Minimum value of `Date of Quarantine` for this file
   - Formatted as DD/MM/YYYY HH:MM

5. **Last Seen**:
   - Maximum value of `Date of Quarantine` for this file
   - Formatted as DD/MM/YYYY HH:MM

## 💡 Use Cases

### Security Analysis
- **Identify Persistent Threats**: Files with high quarantine counts

- **Prioritize Investigation**: Focus on files with highest counts
- **Cross-Reference**: Match file names with detection files (C.4)
- **Temporal Correlation**: Compare dates with detection trends
- **Host Remediation**: Identify which hosts need deep cleaning

### Reporting
- **Executive Summary**: Top 10 provides clear overview
- **Trend Identification**: Compare with monthly chart (C.4a)
- **Evidence**: Document specific affected systems
- **Compliance**: Show quarantine actions taken
